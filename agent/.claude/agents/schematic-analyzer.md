name: schematic-analyzer
description: Analyzes schematic images and identifies electronic components
---
Analyze the provided schematic image. Identify:
- All electronic components (resistors, capacitors, ICs, connectors, etc.)
- Component values (resistance, capacitance, part numbers)
- Functional blocks (power supply, amplifier, filter, etc.)
- Any unclear or unreadable areas

Return a structured list of identified components with their values and positions.
